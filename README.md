This repository contains Martini coarse-grained molecular dynamics simulation videos illustrating the membrane activity of vesicle-embedded pneumolysin during interaction with a target membrane. In the wild-type system (Movie S1), Domain 1 (D1) drives pronounced membrane curvature, highlighting its role in early-stage membrane remodeling prior to pore formation. In contrast, the mutant system (Movie S2), with substitutions at key membrane-interacting residues (L130D, W134D, H135G), shows reduced curvature and weaker deformation, underscoring the importance of these residues in governing membrane remodeling.
